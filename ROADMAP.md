# GovKR-MD Roadmap

이 문서는 프로젝트 개발 계획을 설명합니다.

---

# Phase 1 — Foundation

목표

* 프로젝트 구조 구축
* Markdown 스키마 정의
* 기본 크롤러 개발

포함

* 법령 문서
* 시행령
* 행정규칙

---

# Phase 2 — Expansion

목표

* 정책 문서 수집
* 보고서 아카이브 구축
* 검색 시스템 구축

포함

* 정책 보고서
* 백서
* 공공 연구 보고서

---

# Phase 3 — Automation

목표

* 자동 문서 수집
* 문서 diff 추적
* 검색 API

포함

* 크롤링 자동화
* 버전 비교

---

# Phase 4 — Platform

목표

* 공개 문서 플랫폼 구축
* 시민 기술 도구 지원

가능한 기능

* 정책 diff 시각화
* 문서 그래프
* AI 분석 도구

---

# Long Term Vision

GovKR-MD는 **대한민국 공공 행정 문서의 Markdown 아카이브**를 구축하는 것을 목표로 합니다.

이를 통해 다음이 가능해집니다.

* 정책 변화 추적
* 법령 diff 분석
* 시민 기술 도구 개발
* 공공 정책 연구
