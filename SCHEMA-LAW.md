# GovKR-MD Law Schema

> 한국 법령 Markdown 구조 표준 v1.0
>
> 이 스키마는 [Akoma Ntoso](http://docs.oasis-open.org/legaldocml/akoma-ntoso/) 국제 표준과 한국 법령 체계를 기반으로 설계되었습니다.

---

## 1. 법령 메타데이터 (Frontmatter)

모든 법령은 다음 YAML frontmatter를 포함합니다.

```yaml
---
# === 식별 정보 ===
law_id: LS_2025_12345              # 법령 ID (법제처 기준)
law_serial: 제12345호               # 공포번호
title: 개인정보 보호법              # 법령명
abbreviation: 개인정보보호법         # 약칭

# === 분류 정보 ===
law_type: law                      # law | decree | rule | ordinance
ministry: 행정안전부                 # 소관 부처
category: 정보통신                  # 법령 분류

# === 날짜 정보 ===
promulgation_date: 2011-03-29      # 공포일
enforcement_date: 2011-09-30       # 시행일
last_amendment: 2024-03-20         # 최근 개정일

# === 참조 정보 ===
source_url: https://law.go.kr/...,/LS_2025_12345
pdf_url: https://law.go.kr/...,/LS_2025_12345.pdf
official_gazette: 제17611호         # 관보 호수

# === 구조 정보 ===
total_articles: 78                  # 전체 조문 수
attachments: false                  # 별표/서식 여부

# === 메타 정보 ===
status:现行有效                        # 시행 | 폐지 | 일부개정
version: 2024-03-20
---
```

---

## 2. 법령 유형 (law_type)

| type        | 한국어    | 설명            |
|-------------|----------|-----------------|
| `law`       | 법률      | 국회 제정        |
| `decree`    | 대통령령   | 대통령 제정      |
| `rule`      | 총리령/부령 | 행정부 제정     |
| `ordinance` | 조례      | 지자체 제정      |
| `treaty`    | 조약      | 국제 협정        |
| `constitution` | 헌법   | 헌법            |

---

## 3. 조문 구조 (Article Structure)

### 3.1 기본 계층

한국 법령은 4단계 계층 구조를 사용합니다.

```
조 (Article)
  └ 항 (Paragraph/Clause)
      └ 호 (Item)
          └ 목 (Sub-item)
```

### 3.2 Markdown 표현

```markdown
## 제1조 (목적)

이 법은 개인정보의 보호에 관한 사항을 정함으로써 개인의 자유와 권리를 보호하고,
나아가 개인의 존엄과 가치를 구현하는 것을 목적으로 한다.

## 제2조 (정의)

① 이 법에서 사용하는 용어의 뜻은 다음과 같다.

1. "개인정보"란 살아 있는 개인에 관한 정보로서 다음 각 목의 어느 하나에 해당하는
   것을 말한다.
   가. 성명, 주민등록번호 및 영상 등을 통하여 그 개인을 알아볼 수 있는 정보
   나. 해당 정보만으로는 알아볼 수 없으나 다른 정보와 쉽게 결합하여 알아볼 수
       있는 정보
2. "처리"란 개인정보의 수집·보유·이용·제공·파기 등을 말한다.

② 제1항 이외의 용어의 뜻은 관계 법령 또는 일반적 관행에 따른다.
```

### 3.3 구조 태그 (선택사항)

HTML 없이도 의미론적 구조를 유지하기 위해 다음 규칙을 사용합니다.

| 계층   | 기호      | Markdown 패턴              |
|--------|-----------|---------------------------|
| 조     | 제X조     | `## 제X조 (조문명)`        |
| 항     | 제X항     | `①`, `②`, `③` ...        |
| 호     | 제X호     | `1.`, `2.`, `3.` ...      |
| 목     | 제X목     | `가.`, `나.`, `다.` ...   |
| 마     | -         | `1)`, `2)`, `3)` ...      |

## 4. 개정 이력 (Amendment History)

각 법령은 별도의 개정 이력 섹션을 포함합니다.

```markdown
---

## 개정 이력

| 개정일       | 공포번호  | 개정유형   | 주요내용                    |
|-------------|----------|-----------|----------------------------|
| 2024-03-20  | 제19262호 | 일부개정   | 제30조 신설, 제15조 개정    |
| 2023-09-19  | 제18732호 | 일부개정   | 제24조 제2항 개정           |
| 2020-02-04  | 제17611호 | 전부개정   | 법 전면 개정               |

> 상세 개정 내용은 [법제처 원문](https://law.go.kr)을 참조하십시오.
```

## 5. 참조 법령 (References)

법령 간 상호 참조를 명시합니다.

```markdown
---

## 관련 법령

### 상위 법령
- [[헌법]] 제10조, 제17조

### 관련 법령
- [[개인정보 보호법 시행령]]
- [[정보통신망 이용촉진 및 정보보호 등에 관한 법률]]
- [[신용정보의 이용 및 보호에 관한 법률]]

### 하위 법령
- [[개인정보 보호법 시행규칙]]
```

## 6. 별표 및 서식 (Attachments)

법령의 별표, 서식은 별도 파일로 관리합니다.

### 6.1 파일 구조

```
docs/laws/개인정보보호법/
├ 개인정보보호법.md           # 본문
├ 별표1-개인정보처리방침.md    # 별표
├ 별표2-손해배상기준.md
├ 서식1-과징금부과처분서.md
└ images/
  └ 별표3-도식.png            # 도식/그림
```

### 6.2 본문 내 참조

```markdown
제33조(과징금)의 규정에 의한 과징금의 부과기준은 별표 1과 같다.

> 📎 [별표 1] 과징금 부과기준 (./별표1-과징금부과기준.md)
```

## 7. 특수 조문 처리

### 7.1 부칙 (Addenda)

```markdown
---

## 부칙 <제19262호, 2024.03.20.>

제1조(시행일) 이 법은 공포 후 6개월이 경과한 날부터 시행한다.

제2조(경과조치) 이 법 시행 당시 종전의 규정에 의하여 ...
```

### 7.2 제명 (Deleted Articles)

```markdown
## 제50조 (삭제) <제18732호, 2023.09.19.>

> 🗑️ 이 조항은 삭제되었습니다. (2023.09.19. 시행)
```

### 7.3 개정 표시

```markdown
## 제15조 (개인정보의 처리제한) <개정 2024.03.20.>

① ~~~ (종전 제1항) ~~~
② (신설) 개인정보처리자는 ... 한다.  <신설 2024.03.20.>
```
