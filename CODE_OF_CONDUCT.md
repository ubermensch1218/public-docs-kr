# Code of Conduct

GovKR-MD 커뮤니티는 모두에게 열린 공간입니다.

우리는 다음과 같은 가치를 중요하게 생각합니다.

* 존중
* 협력
* 개방성
* 공공성

---

# 기대되는 행동

* 서로 존중하는 커뮤니케이션
* 건설적인 피드백
* 다양한 의견 수용
* 협력적인 문제 해결

---

# 허용되지 않는 행동

다음 행동은 허용되지 않습니다.

* 개인 공격
* 차별적 발언
* 혐오 표현
* 괴롭힘

---

# 문제 신고

문제가 발생한 경우 프로젝트 유지 관리자에게 연락해주세요.

모든 신고는 신중하게 검토됩니다.

---

# 적용 범위

이 규칙은 다음에 적용됩니다.

* GitHub repository
* Issues
* Pull Requests
* 프로젝트 관련 커뮤니티 공간

---

GovKR-MD는 모두가 안전하게 참여할 수 있는 커뮤니티를 목표로 합니다.
